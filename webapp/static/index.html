Where this data comes from

Source A

The mall's map

The data behind the mall's interactive floor plan. It's fast and it's the only place that knows each store's unit number (like A010). This is our main source.

Source B

The mall's website

The public store list shoppers see. It's the official roster, but it doesn't publish unit numbers. We use it purely to check Source A isn't missing anyone.

What "cross-check" means, and what it can't promise

The check: we pull both lists and compare the store names. If the website lists a store the map doesn't have, we flag it — that's a real gap we've caught before (a store can be leased and listed before it's drawn onto the map).

The honest limit: this only compares names. A clean result does not verify that unit numbers are correct. It also only proves the mall's two systems agree with each other — not that either matches the physical mall today. The mall's site also blocks bots, so the cross-check sometimes can't run; when that happens we say so rather than quietly claiming success.
